A popular science interactive platform for card drawing probability
Core concept: Breaking the misconception that "probability = European luck" - Witnessing the Bernoulli Experiment and the Independence Event

Explanation of Interaction Rules
1. Basic Settings: This program strictly locks the SSR card release probability at 1%.
2. Core concept: Each card draw is a Bernoulli experiment. The result of the previous one will never affect the next one. They are completely independent events.
3. Interactive Process: In the subsequent stages, please follow the steps of "input prediction → hands-on operation verification → comparison with theoretical data".
